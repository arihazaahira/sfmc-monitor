📖 How to Get Your Credentials
Create an Installed Package in SFMC:
  1. Log in to Salesforce Marketing Cloud.
  2. Go to Setup → Apps → Installed Packages.
  3. Click New, give it a name (e.g. SFMC Dashboard), then Save.
  4. Click Add Component → API Integration.
  5. Choose Server-to-Server (no user context needed).
  6. Grant these scopes: contacts_read, data_extensions_read, automations_read.
  7. Save. You'll see your Client Id, Client Secret, and MID.
  8. Your Subdomain is the unique prefix in your tenant's auth URL:
    https://mc123abc.auth.marketingcloudapis.com
🛡️ Security Notes
  • Credentials are stored only in chrome.storage.local on this device.
  • The client_secret is never displayed in the UI after being saved.
  • Access tokens are cached in memory and auto-refreshed before they expire.
  • No data is sent to any third-party server — all requests go directly to *.marketingcloudapis.com.
